Display your miniBB-powered forum's content, like recent and most popular topics, with Widgets.

== Description ==

The plugin has been tested with WordPress 4.7.4 and miniBB 3.2.2, but it should work in later and ealier versions as well.

== Installation ==

Installing this plugin happens like any other WordPress plugin: upload plugin folder to you `/wp-content/plugins/` directory, or install through WordPress plugins screen. Then:

1. Activate the plugin through the 'Plugins' screen in WordPress
2. Use the Settings -> Plugin Name screen to configure the plugin
3. Go to Widgets and you should see there miniBB Widget.
